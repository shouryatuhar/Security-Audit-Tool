# Security Policy

## Supported Versions

Currently, only the latest release of HostSentinel receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within HostSentinel, please do not disclose it publicly. Instead, send an email to security@hostsentinel.example.com. All security vulnerabilities will be promptly addressed.
